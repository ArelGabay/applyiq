# ApplyIQ — AI Job Application Assistant

## Goal

Build a polished portfolio-quality AI-powered ATS resume optimization platform.

Users upload resumes and job descriptions and receive:

- ATS score
- Missing keywords
- Resume suggestions
- AI rewrites
- Tailored cover letters

## Tech Stack

Frontend:

- Next.js
- TypeScript
- Tailwind CSS

Backend:

- FastAPI
- Python

Database:

- PostgreSQL later

AI:

- OpenAI API later

## MVP Priorities

1. Landing page
2. Dashboard UI
3. Resume upload page
4. Analysis results page
5. Mock AI data first

## Avoid For Now

- Authentication
- Payments
- LinkedIn scraping
- Real ATS integrations
- Complex resume builder
- Overengineering

## Development Style

- Keep components simple
- Build fast
- Use mock data first
- Prioritize clean UI
- Make the project look production-ready
